# 📘🏛️ OmniverseCORE
>(v1.0 → v4.0 NextGen)

🎯 Objectif
Décrire l’évolution complète de l’architecture d’OmniverseCORE, depuis la fondation (v1.0) jusqu’à la version NextGen (v4.0), en détaillant les blocs, couches, modules et transformations majeures.

🏗️ v1.0 — Core Minimal
- Architecture MVC simplifiée
- Routing basique
- Layout statique
- Dossiers principaux :
  - /app
  - /public
  - /assets
  - /views
- Documentation initiale

🧱 v1.1 — UI/UX Base
- Ajout du layout global

- Normalisation CSS

- Nettoyage du code
- Préparation modularisation

🧬 v2.9 — Pré‑NextGen

- Préparation v3.0
