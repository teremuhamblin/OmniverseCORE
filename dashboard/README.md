# 🖥️ OmniverseCORE 
>Dashboard
Interface web futuriste pour piloter l’écosystème OmniverseCORE.

🚀 Lancer le dashboard
```bash
npm install
npm run dev
```

🧠 Technologies
- React + TypeScript + Vite
- TailwindCSS
- Axios (API)
- Design cyber‑tech

📡 Connexion API
Le dashboard se connecte à l’API locale :
```text
http://localhost:8000/api/
```

📦 Structure
- src/components — UI modulaire
- src/pages — pages principales
- src/assets — visuels SVG
- public/ — index HTML
